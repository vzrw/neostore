Neostore - نسخة كاملة مستقرة

النسخة هذه مصممة لتعمل محلياً مباشرة بعد فك الضغط، بدون GitHub وبدون سيرفر.
كل شيء يعتمد على localStorage داخل المتصفح لتجنب مشاكل عدم تحميل JS أو JSON.

الملفات:
- index.html
- category.html
- product.html
- cart.html
- admin.html
- assets/*

مواصفات النسخة:
- ثيم فاخر أحمر وأسود
- واتساب وإنستغرام
- أقسام ومنتجات جاهزة
- أدمن مخفي عن الزوار
- الرمز الافتراضي للأدمن: 123456
- إضافة/تعديل/حذف أقسام
- إضافة/تعديل/حذف منتجات
- رفع صورة من الجهاز للمنتج
- تصدير/استيراد JSON

مهم:
- افتح admin.html للدخول للأدمن
- افتح index.html للمتجر
- إذا عدلت من الأدمن فالتعديل يحفظ في نفس المتصفح
- يمكن أخذ نسخة احتياطية من زر التصدير داخل الأدمن
