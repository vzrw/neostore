نسخة GitHub-sync

هذه النسخة تبقي شكل الموقع كما هو تقريباً، لكن صفحات الزوار تقرأ البيانات من الملف:
site-data.json

الطريقة الصحيحة للتحديث:
1) عدل من admin.html
2) اضغط تصدير البيانات JSON
3) ارفع site-data.json إلى جذر المستودع في GitHub
4) Commit changes

بعدها سيرى كل الزوار التغييرات.

مهم:
- هذه النسخة لا تجعل GitHub يستقبل التعديلات مباشرة من الأدمن بشكل آلي.
- النشر التلقائي الحقيقي لكل الزوار من داخل الأدمن يحتاج Firebase أو Backend.
